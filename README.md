# 🎯 Rummy Timer

專為 Rummy 紙牌遊戲設計的專業計時器 iOS 應用程式

## ✨ 功能特色

- ⏰ **智慧計時系統** - 支援 30-300 秒自訂計時
- 👥 **多人遊戲支援** - 支援 2-8 名玩家，自動輪換
- 📊 **詳細統計分析** - 記錄每局遊戲時間、個人用時統計
- 🎨 **精美界面設計** - 支援亮色/暗色主題
- 🔔 **多重提醒系統** - 聲音、震動、視覺警告
- 🌍 **多語言支援** - 繁體中文、簡體中文、英文
